## 💻 Run Locally

1. Clone the repository:

```bash
git clone https://github.com/littleorbitworld0905/amazon-picks.git
```

2. Open the project in **Visual Studio Code**.

3. Install the **Live Server** extension.

4. Right-click on `index.html`.

5. Select **Open with Live Server**.

---

## ➕ Adding a New Product

Open `products.json` and add a new product object.

Example:

```json
{
  "id": 6,
  "name": "Magnetic Cable Organizer",
  "category": "Accessories",
  "price": "₹349",
  "rating": 4.8,
  "added": "2026-07-20",
  "affiliateLink": "https://amzn.to/yourAffiliateLink"
}
```

Save the file, commit your changes, and push them to GitHub. Your live website will update automatically.
